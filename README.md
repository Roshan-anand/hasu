# ! Project is under development, v0.0.1 will release soon

## Tech stack

- **Server:** Go
- **Runtime:** Docker
- **Ingress:** Traefik
- **DB:** SQLite (metadata), BadgerDB (logs)
- **UI:** sveltekit (SPA, served from binary)
