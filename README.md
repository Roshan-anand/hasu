# Godploy

Lightweight, single-binary, self-hosted alternative to Dokploy, Coolify, Netlify, Vercel.

## Tech stack

- **Server:** Go
- **Runtime:** Docker
- **Ingress:** Traefik
- **DB:** SQLite (metadata)
- **UI:** sveltekit (SPA, served from binary)
